# 🎓 Sistema de Matrículas Académicas y Facturación en Go

Este proyecto consiste en una **API REST** desarrollada en **Golang** diseñada para gestionar de manera eficiente el proceso de matriculación estudiantil y la generación automática de facturas digitales. El sistema destaca por su ligereza, alta concurrencia y manejo nativo de datos en formato JSON.

## 🚀 Características Principales

* **Arquitectura REST:** Implementación de métodos HTTP (GET/POST) para la gestión de recursos.
* **Alta Concurrencia:** Aprovecha las *goroutines* de Go para manejar múltiples peticiones simultáneas.
* **Zero Dependencies:** Desarrollado utilizando únicamente la librería estándar de Go (`net/http`).
* **Facturación en Tiempo Real:** Generación automática de comprobantes con ID único y marca de tiempo.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Go (Golang) 1.20+
* **Entorno:** Localhost
* **Formato de Datos:** JSON
